记——将.ts文件合并为mp4文件

19.06.01_02:07:19

ffmpeg   -f  concat   -safe  0 -i   filelist.txt   -c  copy   output.mp4


filelist.txt是文本文件，名字不重要，里面储存了所有ts文件的路径，格式必须如下：

file  ‘第1个ts文件路径’  

file  ‘第2个ts文件路径’  

file  ‘第...个ts文件路径’  

file  ‘最后一个ts文件路径’  


路径的引号是必须的。
